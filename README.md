# limpid-solutions.github.io

Company website for Limpid Solutions.

This is a GitHub Pages site hosted at [limpid-solutions.github.io](https://limpid-solutions.github.io).

## Setup

This site is built with static HTML. To make changes:

1. Edit the HTML files in the repository.
2. Commit and push to the `main` branch.
3. GitHub Pages will automatically deploy the changes.

## Local Development

You can preview the site locally by opening `index.html` in a web browser.
